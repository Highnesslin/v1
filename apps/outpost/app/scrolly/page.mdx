## ! top-36 left-56 w-60

Code Hike will process anything inside a `<Hike/>` component and pass it to a custom layout component.

```mdx
{/* !Mark(1,7:10) */}
{/* !Mark(6[1:5,20]) */}
{/* !from ./snippets/00.hike.mdx */}
```

```jsx
// !Mark(2)
// !from ./snippets/00.hike.jsx
```
